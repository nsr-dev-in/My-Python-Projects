# 🎯 Number Guessing Game (Python)

An interactive command-line **Number Guessing Game** built using Python.  
This beginner-friendly project demonstrates fundamental Python programming concepts such as loops, conditionals, user input handling, and random number generation.

---

## 📖 Overview

In this game, the computer randomly selects a number within a specified range, and the player must guess the correct number.

After each guess, the program provides helpful hints:
- 📈 Too High
- 📉 Too Low
- ✅ Correct Guess

The game continues until the player successfully guesses the number.

This project is ideal for beginners who are starting their Python development journey.

---

## 🚀 Features

- 🎲 Random number generation using Python's built-in `random` module
- ⌨️ Interactive user input
- 🔁 Continuous gameplay using loops
- 🧠 Smart hints after every guess
- 📊 Attempt counter to track performance
- 💻 Simple and beginner-friendly code structure

---

## 🛠️ Technologies Used

- Built-in **random** module

---

## 🎮 How to Play

1. Run the Python script.
2. Enter a number within the given range.
3. The program will guide you:
   - "Too High" if your guess is larger.
   - "Too Low" if your guess is smaller.
4. Keep guessing until you find the correct number.
5. The game will display the total number of attempts taken.
