# 🎯 Number Guessing Game (Python)

A simple interactive **Number Guessing Game** built using Python.  
This beginner-friendly project helps you understand core Python concepts like random number generation, user input handling, loops, and conditional statements.

---

## 📌 Project Description

In this game, the computer randomly selects a number within a given range, and the player tries to guess it.  
The program provides hints such as **"Too High"** or **"Too Low"** until the correct number is guessed.

This project is perfect for beginners who are starting their Python journey.

---

## 🚀 Features

- 🎲 Random number generation
- ⌨️ User input handling
- 🔁 Continuous guessing using loops
- 🧠 Conditional logic for hints
- 📊 Attempt counter
- ✅ Winning message when guessed correctly

---

## 🛠️ Technologies Used

- Python 3
- `random` module (built-in Python library)
