# 🔗 Streamlit URL Shortener

A simple and interactive web application built using **Python and Streamlit** that allows users to instantly convert long URLs into short, shareable links directly from their browser. This project demonstrates how Python alone can be used to create a live web application without traditional frontend frameworks, making it perfect for beginners learning web app development. The application provides a clean interface where users can enter a long URL, click a button, and receive a shortened link generated using the TinyURL service through the `pyshorteners` library. It includes features such as instant URL shortening, simple user interaction, browser-based execution, warning messages for empty inputs, and a lightweight responsive UI powered entirely by Streamlit. The project is built using **Python** as the main language, **Streamlit** for the web interface, and **pyshorteners** for API-based URL shortening. The project structure is minimal and beginner-friendly: `streamlit-url-shortener/` containing `app.py` (main application file) and `README.md`. To set up the project, clone the repository using `git clone https://github.com/your-username/streamlit-url-shortener.git`, navigate into the folder, optionally create a virtual environment using `python -m venv venv`, activate it, and install dependencies with `pip install streamlit pyshorteners pyperclip`. Run the application using `streamlit run app.py`, then open `http://localhost:8501` in your browser to access the live app. To use the application, simply enter a long URL, click **Shorten URL**, and instantly receive a shortened version ready for sharing. Through this project, key concepts learned include building interactive apps with Streamlit, handling user inputs, working with APIs, understanding event-driven UI logic, and deploying Python scripts as web applications. Example usage includes converting a long link such as `https://www.example.com/some/very/long/link` into a compact TinyURL link for easier sharing. Future improvements may include URL validation, copy-to-clipboard functionality, link analytics, database storage for history, enhanced UI styling, authentication, custom short codes, and deployment on Streamlit Cloud.
